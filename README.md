# 7on7 Playbook Handoff Package

This directory is a sanitized handoff copy of the original `7on7` app.

It includes:
- The runtime web app and Docker files
- A safe starter `playbook.db` with playbook data only

It does not include:
- Real team roster or contact data
- OAuth tokens, client secrets, or spreadsheet integration files
- Private TLS keys or local deployment scripts
- Local tooling folders and one-off maintenance scripts

Setup starts in [SETUP.md](./SETUP.md).
